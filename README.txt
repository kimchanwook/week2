Compiling and Running bubblesort.c

1. Compiling

gcc -std=c99 bubblesort.c -o bubblesort

2. Run the program

./bubblesort

When it asks you "Enter number of numbers",
type in count of numbers you wish to sort up to 100.

It will ask you to "Enter # numbers".
Then, enter numbers with spaces inbetwee.
The numbers will be sorted in ascending order.
